The game contains 2 modes: Manual AI mode (genetic Algorithm mode) Player can choose to use the game in any of given modes. Reversed Stable diffusion

Reverse the typical direction of a generative text-to-image model: instead of generating an image from a text prompt, creating a model which can predict the text prompt given a generated image. I made predictions on a dataset containing a wide variety of (prompt, image) pairs generated by Stable Diffusion 2.0, in order to understand how reversible the latent relationship is.
